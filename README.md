# minecraft-panel-web
Minecraft Server Panel — это веб-панель управления для сервера Minecraft, написанная на Python с использованием Flask. Панель позволяет запускать, останавливать сервер, отправлять команды в консоль, просматривать логи в реальном времени, управлять плагинами (включая поиск и скачивание с Modrinth), создавать резервные копии мира и следить за системными ресурсами.
Основные возможности

    Запуск/остановка сервера – управление через веб-интерфейс.

    Веб-консоль – отправка команд и цветное отображение логов.

    Список онлайн-игроков – автоматическое обновление.

    Управление плагинами – загрузка, удаление, интеграция с Modrinth (поиск и установка).

    Резервное копирование – создание и восстановление архива мира.

    Мониторинг – графики загрузки CPU и RAM (статистика сервера).

    Системные ресурсы – отображение использования CPU, RAM, диска.

    Мультиязычность – поддержка русского и английского языков.

    Современный дизайн – тёмная тема, адаптивная вёрстка, анимации.

Технологии

    Backend: Python 3, Flask, psutil, requests, PyInstaller (для сборки)

    Frontend: Bootstrap 5, Chart.js, HTML/CSS/JavaScript

    API: Modrinth API для поиска и скачивания плагинов
