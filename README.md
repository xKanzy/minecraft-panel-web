# minecraft-panel-web
Minecraft Server Panel — это веб-панель управления для сервера Minecraft, написанная на Python с использованием Flask. Панель позволяет запускать, останавливать сервер, отправлять команды в консоль, просматривать логи в реальном времени, управлять плагинами (включая поиск и скачивание с Modrinth), создавать резервные копии мира и следить за системными ресурсами.

## Дисклеймер!
Проект сырой и в разработке поэтому могут быть проблемы.

## Особенности

- Управление сервером (запуск, остановка)
- Веб-консоль с цветным выводом
- Список онлайн игроков
- Загрузка/удаление плагинов (также через Modrinth)
- Создание и восстановление резервных копий мира
- Мониторинг системных ресурсов
- Поддержка русского и английского языков

## Установка и запуск исходников

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python app.py`
4. Откройте в браузере: `http://localhost:5000`

## Настройка

При первом запуске откроется страница настройки, где нужно указать путь к папке с сервером Minecraft.
