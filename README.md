# minecraft-panel-web
Minecraft Server Panel — это веб-панель управления для сервера Minecraft, написанная на Python с использованием Flask. Панель позволяет запускать, останавливать сервер, отправлять команды в консоль, просматривать логи в реальном времени, управлять плагинами, создавать резервные копии мира и следить за системными ресурсами.
